Admin: User Management

Important Instructions for User Management:

  • This page allows administrators to manage user profiles within ShutterSync CRM.
  • **Recommended Workflow for New Users:**
    • Ask new users to **first log in to the application** using their Google account (or other configured authentication method). This automatically creates their basic user entry in Firebase Authentication.
    • Once they have logged in at least once, an administrator can then come to this "User Management" page.
    • Find the user in the "Existing Users" list (they'll likely have the default 'User' role).
    • Click "Edit" to assign them a specific role (e.g., 'Admin') and add any other profile details.
  • **Adding a User Profile Manually (Advanced):** If you need to add a user profile here *before* they log in (e.g., for specific setup scenarios), you **MUST** first create the user in your Firebase Console (Authentication section). Then, copy their unique **User UID** and paste it into the "User Firestore ID (UID)" field when creating their profile below.
  • For existing users, the "User Firestore ID (UID)" field will be read-only during editing.

Add New User Profile
