Admin: User Management

Important Instructions for User Management:

  • This page is for administrators to manage user profiles in the system.
  • To create a new user profile here, the user **MUST first be created in Firebase Authentication**. Go to your Firebase Console > Authentication, then add a new user (e.g., via Email/Password or Google). Once created, copy their unique **User UID**.
  • Paste the Firebase User UID into the "User Firestore ID (UID)" field below when creating a new profile.
  • Set the user's role to 'Admin' to grant them administrative access to CRM admin features.
  • For existing users, their UID field will be read-only during editing.
