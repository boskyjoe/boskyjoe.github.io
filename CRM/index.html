Admin: Country and State Mapping

WARNING: This tool directly modifies your Firebase Firestore `app_metadata/countries_states` document.
For this tool to work, you MUST temporarily update your Firestore Security Rules to allow write access to `app_metadata/{document=**}` for your authenticated user. Remember to revert rules after use for security.

Manage Country Data

Example: `United States,US;Canada,CA;Germany,DE`

Example: `US:Alabama,Alaska;CA:Ontario,Quebec`

Load Type:

* Full Load will replace the entire "countries" and "countryStateMap" fields. * Incremental Load will merge the top-level fields. If a textarea is left blank, that specific field will not be updated.
